Server setup

Create droplet

Distribution: Fedora31 x64

Plan: Standard

Datacenter region: Amsterdam

Additional options: Monitoring

Authentication: SSH keys

Add user

# Add a new user so that you're not always in root

useradd -m -G wheel plien

# Give new user a password

passwd plien

# Switch from root to your new user

su plien

Move SSH keys

# Copy the ssh keys. Sudo because you need to be in root to acces it and the dot at the end means 'here'

sudo cp /root/.ssh/authorized_keys .

# Make a .ssh directory

mkdir .ssh

# Move the authorized_keys to the .ssh folder

mv authorized_keys .ssh/

Change ownership permissions and delete keys from root

# Change ownership permission to user plien and group plien

sudo chown plien:plien .ssh/authorized_keys

# Make sure only plien can acess and not other users or outsiders

chmod 700 .ssh

# Remove the authorized_keys file from the root account to prevent being able to ssh directly into root

rm /root/.ssh/authorized_keys
